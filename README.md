# 📚 Collaborative Study Planner

**Live Deployment:** [collaborative-study-planner.web.app](https://collaborative-study-planner.web.app/)

## Overview
The Collaborative Study Planner is a full-stack, real-time workspace designed to help student teams organize assignments, share resources, and communicate effectively. Built with React and Firebase, it combines task management with built-in productivity tools like a Pomodoro timer and live chat.

## ✨ Key Features
* **Real-Time Task Management:** Create, assign, and track tasks with custom due dates and priority levels (High/Medium/Low).
* **Live Team Chat:** A persistent, real-time chat widget for seamless communication without leaving the workspace.
* **File Sharing:** Securely upload and download assignment files, PDFs, and reference materials using Firebase Storage.
* **Built-in Focus Timer:** An integrated Pomodoro-style timer to help maintain focus during deep-work sessions.
* **User Profiles & Analytics:** Personalized dashboards tracking individual task completion statistics.
* **Dynamic Theming:** Seamless Dark and Light mode toggle for optimal viewing in any environment.
* **Secure Authentication:** Google OAuth integration ensuring only authorized team members can access the workspace and database.

## 🛠️ Tech Stack
* **Frontend:** React, Vite, Tailwind CSS, Lucide React (Icons)
* **Backend & Database:** Firebase Firestore (Real-time NoSQL)
* **Authentication:** Firebase Auth (Google Provider)
* **File Storage:** Firebase Storage
* **Hosting:** Firebase Hosting
